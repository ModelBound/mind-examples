# customer-support-agent

A `.mind/` starter for a support agent that routes questions to product docs, tone guides, and customer history.

```
.mind/
├── INDEX.md
├── self.md
├── context/
│   ├── product.md
│   └── tone.md
└── skills/
    └── triage.md
```

Copy `.mind/` into your project and point your agent (Cursor, Claude, custom) at it.
